
## docker-elasticsearch

##  安装并启动

1. 创建data和logs目录

```
mkdir -p es01/data
mkdir -p es01/logs
mkdir -p es02/data
mkdir -p es02/logs
mkdir -p es03/data
mkdir -p es03/logs

## es的用户id为1000，这里暂且授权给所有人好了
sudo chmod 777 es* -R

```

2. 启动

```
sudo docker-compose up

```
